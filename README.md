# 🌌 MH2 Galaxy Engine

A realistic real-time 3D Solar System and Space Simulation built using Three.js and JavaScript.

---

## 🚀 Features

- ☀️ Realistic Sun with Glow Effect
- 🪐 Animated Planet Orbit System
- 🌍 Earth + Moon Simulation
- ✨ Massive Space Starfield
- 🎮 WASD Free Camera Movement
- 🖱️ Mouse Drag Camera Rotation
- 💡 Dynamic Lighting System
- 🔥 Cinematic 3D Text Rendering
- ⚡ Real-time Rendering with WebGL
- 🌌 Immersive Space Environment

---

## 🎮 Controls

| Key | Action |
|------|--------|
| W A S D | Move Camera |
| Mouse Drag | Look Around |
| Arrow Up | Fly Up |
| Arrow Down | Fly Down |
| Arrow Left | Rotate Left |
| Arrow Right | Rotate Right |
| 1 / 4 | Camera Pitch |
